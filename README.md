# kicad_lib
Personal Kicad library
======================

This library has been used mainly for:
- help with proof of concept (to draw circuits using modules before wiring on the breadboard)
- add abstract models to simulations using Kicad + ngspice (I'm still to add PMAC/BLCD motor model and some samples of use)
